# Credit Application Documents

This directory contains downloadable forms and templates for credit applications.

## Required Files

### BH-Demande-Credit-Habitat.pdf
- **Description**: BH Habitat Credit Request Form (Demande de Crédit Habitat)
- **Purpose**: Official form from BH Bank for credit applications
- **Status**: Should be placed in this directory
- **URL**: `/documents/BH-Demande-Credit-Habitat.pdf`

## Setup Instructions

1. Obtain the official BH Bank credit application form from the bank
2. Save it as `BH-Demande-Credit-Habitat.pdf` in this directory
3. The file will be automatically served to clients for download before they submit their applications

## Document Types

The credit application system supports the following document types that users must upload:

- **BH_FORM**: BH Habitat Form (required)
- **ID_COPY**: Identity Document - CIN/Passport copy (required)
- **INCOME_PROOF**: Income Proof - Pay stubs, tax returns (required)
- **PROPERTY_DOCS**: Property Documents - Deed, survey, etc. (required)
- **BANK_STATEMENTS**: Bank Statements - Last 3-6 months (required)
- **EMPLOYMENT_CONTRACT**: Employment Contract or professional status proof (required)
